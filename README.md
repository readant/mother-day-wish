# 母亲节祝福互动礼物

一个具有视觉吸引力和情感表达的母亲节主题互动礼物网页。

## 功能特性

- 🎀 **蝴蝶结启动画面** - 精致的立体蝴蝶结设计，带有柔和的粉色渐变色彩和浮动动画
- 💕 **点击触发交互** - 用户点击蝴蝶结触发后续动画序列
- 💓 **心形绘制动画** - 手绘风格的渐进式心形绘制效果，模拟铅笔勾勒心形轮廓
- 💗 **心形动态效果** - 心形完成后上下跳动，然后逐渐放大至全屏
- 🌸 **缤纷心形粒子** - 从屏幕顶部飘落的五彩缤纷小心形粒子效果
- 💐 **康乃馨展示** - 精美的康乃馨花朵 SVG 绘制
- 📝 **温馨祝福语** - 依次显示母亲节祝福语句

## 技术栈

- HTML5
- CSS3 (动画、渐变、SVG)
- JavaScript (原生)

## 使用方式

### 本地运行

```bash
# 方式1：使用 Python HTTP 服务器
python -m http.server 8080

# 方式2：使用 Node.js HTTP 服务器
npx http-server -p 8080

# 方式3：直接双击打开 index.html
```

打开浏览器访问 http://localhost:8080

### Gitee Pages 部署

1. 创建 Gitee 仓库
2. 将项目文件推送到仓库
3. 进入仓库设置，开启 Gitee Pages
4. 选择 `master` 分支，`/` 目录
5. 点击"启动"按钮

## 动画流程

1. **初始界面** - 显示蝴蝶结和提示文字
2. **点击蝴蝶结** - 蝴蝶结打开动画，提示文字隐藏
3. **心形绘制** - 手绘风格心形轮廓绘制
4. **心形跳动** - 心形上下跳动动画
5. **心形放大** - 心形逐渐放大至全屏并淡出
6. **粒子飘落** - 五彩缤纷的心形粒子从顶部飘落
7. **康乃馨显示** - 康乃馨花朵淡入显示
8. **祝福语展示** - 温馨祝福语依次显示

## 设计理念

- **色彩搭配**：采用温馨的粉色调，传达温暖和爱意
- **动画效果**：流畅自然的过渡动画，营造浪漫氛围
- **情感表达**：通过视觉元素传递对母亲的感恩与祝福
